Spécificités du projet

Fonctionnalités

.

Construit à l'aide du cadre moderne HTML5, CSS3 et jQuery avec prise en charge de tous les navigateurs modernes, y compris Edge, Chrome, Firefox, ... (et Internet Explorer). Veuillez passer en mode plein écran pour une meilleure expérience.

SASS Préprocesseur

CSS3 a été codé via le préprocesseur CSS SASS. De plus, le projet comporte une utilisation intensive de CSS3 et de ses fonctionnalités. Vous trouverez des info-bulles CSS personnalisées, des menus contextuels modifiés, des transformations sympas, des transitions fluides et tout un tas de beaux composants.

jQuery complète la magie

Utilisation de jQuery pour contrôler l'ensemble de la séquence et pour intégrer des fonctionnalités dans ce projet.

Faites glisser et déposer !

Utilise le plug-in jQuery UI Draggable pour permettre le réarrangement des mosaïques de l'interface utilisateur métropolitaine et des icônes du bureau et pour faire glisser les applications de fenêtre.

Icons lui donne vie !

Toutes les icônes (à l'exception de l'icône Brackets) ont été servies à partir de la bibliothèque Ionicons.

Et les smartphones ?

Tous les composants ont été soigneusement dimensionnés pour s'adapter à un environnement réactif. Par conséquent, Windows 8.1 pour le Web fonctionne aussi bien même lorsque vous effectuez un zoom avant ou même si vous visualisez ce projet sur votre smartphone ou votre tablette.

Ecran de démarrage

Le Pacejs s'est intégré à l'écran de démarrage, de sorte que le processus de démarrage ne se termine qu'une fois que tout est chargé. Cela amène l'expérience utilisateur à de plus hauts sommets.
